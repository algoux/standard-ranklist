# 格式转换工具

## 简介

[Standard Ranklist Convert-to](https://github.com/algoux/standard-ranklist-convert-to) 是一个格式转换工具，可以将 srk 数据转换为其他平台和工具所需的格式。

## 支持的目标格式

| 目标格式 | 说明 |
|---------|------|
| Codeforces Gym | Codeforces Gym 训练赛格式 |
| Excel | 电子表格格式，方便打印和分发 |
| VJ Replay | Virtual Judge 回放格式 |

更多格式正在持续添加中。

## 适用场景

- 将 srk 格式的排名数据导出为 Excel 供线下使用
- 将排名数据导入 Codeforces Gym 创建训练赛或重现赛
- 导入生成 VJ 比赛回放
