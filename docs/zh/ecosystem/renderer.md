# Renderer 渲染组件

## 简介

[Standard Ranklist Renderer Component](https://github.com/algoux/standard-ranklist-renderer-component) 是一个现代化的 Web 渲染组件，可以将 srk 数据渲染为美观、交互式的排名表格。

## 应用场景

Renderer 组件已被广泛应用于：

- 过往多届 CCPC 官方榜单页面
- [RankLand](https://rl.algoux.org/) 平台的榜单展示
- Kessoku Series 直播导播套件
- 其他社区开发的竞赛相关应用

## 快速使用

::: tip 在线体验
最快的体验方式是访问 [RankLand Playground](https://rl.algoux.org/playground)，无需安装任何依赖即可看到 Renderer 的渲染效果。
:::

如需在自己的项目中使用，请参阅项目仓库的 README 获取安装和配置指南：

→ [algoux/standard-ranklist-renderer-component](https://github.com/algoux/standard-ranklist-renderer-component)
