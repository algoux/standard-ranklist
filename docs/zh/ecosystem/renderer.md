# Renderer 渲染组件

## 简介

[Standard Ranklist Renderer Component](https://github.com/algoux/standard-ranklist-renderer-component) 是一组现代化的 Web 渲染组件，可以将 srk 数据渲染为美观、交互式的排名表格。它面向 React、Vue、Angular、Solid 和 Svelte 发布框架专用组件包，并复用共享的 core 与 styles 包。

## 应用场景

Renderer 组件已被广泛应用于：

- 过往多届 CCPC 官方榜单页面
- [RankLand](https://rl.algoux.cn/) 平台的榜单展示
- Kessoku Series 直播导播套件
- 其他社区开发的竞赛相关应用
